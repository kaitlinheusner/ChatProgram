# Chat Program

It is a chat application that allows clients to connect to a server and send/receive messages. 

## Instructions for Build and Use

Steps to build and/or run the software:

1. Open VS Code
2. Open multiple terminals and run 'server.py' first then as many clients you want using 'client.py.' You may have to use py or python3 before doing this.  
3. Interact using the 'client.py' and look at the 'server.py' to see what is happening overall. 

Instructions for using the software:

1. Open multiple terminals and run 'server.py.' 
2. Run as many clients as you want with 'client.py.'
3. Communicate using the 'client.py' and look at the server terminal to get an idea of the overall scope of what is happening in the program.
